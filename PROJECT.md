## 部署步骤

### 1. 下载项目
```bash
# 项目位置
/workspace/wechat-lottery-miniprogram/
```

### 2. 导入微信开发者工具
1. 打开微信开发者工具
2. 选择「导入项目」
3. 选择项目目录
4. 填写 AppID

### 3. 配置 AppID（可选）
如果要使用完整功能，需要在微信公众平台注册小程序并获取 AppID

### 4. 真机调试
- 点击「预览」生成二维码
- 使用微信扫码体验

### 5. 发布上线
1. 点击「上传」提交代码
2. 在微信公众平台提交审核
3. 审核通过后发布

## 注意事项

### 微信平台限制

1. **群聊功能**
   - 小程序无法直接获取群列表
   - 需要用户分享到群后才能获取群ID
   - 当前版本使用模拟数据演示

2. **好友列表**
   - 小程序无法获取好友列表
   - 建议使用手动添加功能

3. **本地存储**
   - 单个小程序最多10MB存储
   - 历史记录会自动清理（保留最近50条）

### 真实数据接入

要接入真实的微信群数据，需要：

1. 配置分享功能
```javascript
// 在页面中添加分享配置
onShareAppMessage() {
  return {
    title: '邀请你参与抽奖',
    path: '/pages/index/index',
    success: (res) => {
      // 获取群ID
      console.log(res.shareTickets)
    }
  }
}
```

2. 使用群ID获取成员信息
```javascript
wx.getGroupChatInfo({
  groupId: groupId,
  success: (res) => {
    // 处理群成员数据
  }
})
```

## 常见问题

### Q: 如何修改主题色？
A: 修改 `app.wxss` 中的颜色变量

### Q: 如何添加云存储？
A: 在 `app.js` 中初始化云开发，并修改存储逻辑

### Q: 如何添加更多奖项？
A: 点击「+ 添加奖项」按钮，支持无限添加

### Q: 抽奖算法是否公平？
A: 使用 Fisher-Yates 洗牌算法，保证随机性
