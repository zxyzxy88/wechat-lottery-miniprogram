# 微信抽奖小程序

## 功能特点

✨ **多种参与者来源**
- 📱 微信好友列表选择
- 👥 微信群全部成员
- 🔍 微信群部分成员筛选
- ✏️ 手动添加参与者

🎯 **灵活的奖项设置**
- 自定义奖项名称和数量
- 可视化奖项等级显示
- 支持多个奖项同时抽奖

🎊 **精彩的抽奖动画**
- 滚动名字动画效果
- 中奖者揭晓动画
- 全屏沉浸式体验

📊 **历史记录管理**
- 自动保存抽奖记录
- 查看历史中奖名单
- 支持导出和分享

## 界面设计

采用微信风格设计语言：
- 🎨 绿色主题色调（微信品牌色）
- 📐 清晰的信息层级
- 🖱️ 流畅的交互动画
- 📱 响应式布局（支持手机和电脑）

## 使用说明

### 1. 选择参与者
点击不同的数据来源选项：
- **微信好友**：打开好友列表，选择要参与抽奖的好友
- **群成员（全部）**：选择一个群，自动添加所有成员
- **群成员（部分）**：选择群后，筛选要参与的成员
- **手动添加**：直接输入参与者姓名

### 2. 设置奖项
- 点击"添加奖项"创建新奖项
- 设置奖项名称和名额
- 可随时修改或删除奖项

### 3. 开始抽奖
- 点击"开始抽奖"按钮
- 系统会依次抽取各个奖项的获奖者
- 抽奖过程有精彩动画展示

### 4. 查看结果
- 抽奖完成后自动显示中奖名单
- 可保存到历史记录
- 在"历史记录"标签页查看过往记录

## 技术说明

- 纯前端实现，无需后端服务
- 使用 LocalStorage 存储历史记录
- 模拟数据用于演示功能
- 可轻松接入真实微信数据接口

## 开始使用

直接在浏览器中打开 `index.html` 即可使用！

或启动本地服务器：
```bash
cd lottery-app
npx http-server -p 8080
```

然后访问 http://localhost:8080

---

**注意**：当前版本使用模拟数据演示功能。如需接入真实微信数据，需要：
1. 在微信小程序环境中运行
2. 调用微信开放接口获取好友/群组数据
3. 遵守微信平台的相关规定
